# React.js Frontend with GraphQL

## 📦 Installation

### Requirements

Make sure the following are installed on your system:

- **Node.js** (version 18+ recommended; avoid unstable versions like v25)
- **npm** (comes with Node.js)

### Environment Variables

Create a `.env` file in the root of the project and define the backend URL:

```env
VITE_BACKEND_URL=http://localhost:8080/api/graphql
```

> Note: In Vite, environment variables must be prefixed with `VITE_` to be exposed to the client.

### Install Dependencies

Run the following command:

```bash
npm install
```

## 🚀 Running the Application

Start the development server:

```bash
npm run dev
```

The app will typically be available at:

```
http://localhost:5173
```
